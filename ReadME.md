# Drupal Travel Guide Project 
 
Built as a GSoC 2026 preparation project to demonstrate Drupal 11 expertise. 
 
### Key Technical Features: 
- **Entity Relationships**: Linked Activities to Destinations using Entity References. 
- **Advanced Views**: Created dynamic blocks with Contextual Filters. 
- **Layout Builder**: Implemented drag-and-drop page architecture. 
- **Configuration Management**: Fully exported site settings to YAML.
